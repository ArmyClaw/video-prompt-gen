# 🎬 Video Prompt Gen — AI视频提示词生成器

> 专业级AI视频提示词编排工具。支持相机运动、电影风格、镜头语言、角色动作等全套参数。

## 阶段规划

### Phase 1: 核心参数体系
- [ ] 相机类型（推拉摇移跟升降）
- [ ] 镜头角度（平角/仰角/俯角/鸟瞰/荷兰角）
- [ ] 运动速度（缓慢/中速/快速/变速）
- [ ] 景别（远景/全景/中景/近景/特写）

### Phase 2: 电影风格系统
- [ ] 电影流派（科幻/黑色电影/纪录片/anime/武侠）
- [ ] 灯光风格（自然光/霓虹/戏剧性/柔光/逆光）
- [ ] 调色方案（暖调/冷调/去饱和/高对比）
- [ ] 胶片质感（胶片颗粒/数码锐利/复古VHS）

### Phase 3: 场景与角色
- [ ] 角色动作库（行走/奔跑/回望/转身等）
- [ ] 环境描述（室内/室外/城市/自然/抽象）
- [ ] 道具与互动
- [ ] 情绪与氛围

### Phase 4: 高级编排
- [ ] 多镜头序列编排（分镜表）
- [ ] 镜头转场设计
- [ ] 时间线编辑器
- [ ] 预设模板库

### Phase 5: 平台适配
- [ ] Runway ML 提示词格式
- [ ] Pika Labs 格式
- [ ] Kling (可灵) 格式
- [ ] Sora 格式
- [ ] 自定义格式导出

## 技术选型
- **前端**: 纯 HTML/CSS/JS
- **后端**: 无需（纯客户端生成）
- **部署**: 静态页面
